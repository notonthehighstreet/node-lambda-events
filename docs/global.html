lambdaCallback(res, bodyopt)

A Lambda Callback signals to AWS Lambda that the function has completed either successfully or otherwise. It can contain a result body (as a string) or throw an Error object.
Parameters:
Name Type Attributes Default Description
res null | Error The result of the function being called
body string <optional>
